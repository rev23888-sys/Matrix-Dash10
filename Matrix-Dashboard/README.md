# Matrix Bot Dashboard

A modern, feature-rich web dashboard for **Matrix Bot** — built with Next.js and Chakra UI.

## Features

- 🛡️ **Moderation** — Ban, kick, mute, warn and manage members
- 🚨 **Anti-Nuke** — Protect your server from raids and nukes
- 🤖 **AutoMod** — Auto-filter spam, links, caps and bad words
- ⭐ **Levels & Economy** — Global XP system with leaderboards
- 🎵 **Music Player** — YouTube, Spotify & SoundCloud support
- 🔊 **Voice Master** — Custom temporary voice channels
- 👋 **Welcome Messages** — Custom welcome cards for new members
- 📊 **Invite Tracker** — Track who invited who
- 💬 **Auto React** — Auto-react to messages with custom emojis
- 🎭 **Role Manager** — Advanced role assignment and menus

## Setup

1. Copy `.env.example` to `.env` and fill in your values
2. Install dependencies: `pnpm install`
3. Run dev server: `pnpm dev`
4. Build for production: `pnpm build`
